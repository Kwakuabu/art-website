Internal Research

Overview

ART Ghana conducted original computer vision research using the Brazilian Retinal Image Dataset (BRSET) — a publicly available benchmark dataset for diabetic retinopathy grading and ophthalmic condition classification.

The research explored deep learning model architectures for automated retinal lesion detection, contributing to the broader goal of accessible diagnostic intelligence for healthcare settings in under-resourced regions.

Technical scope

  • check Convolutional neural network (CNN) architectures for multi-class retinal grading
  • check Transfer learning from pre-trained vision models for efficient training on limited labelled data
  • check Cloud-based model training using GPU compute on AWS
  • check Evaluation against standard ophthalmology grading benchmarks
